Configuring Service Data Agent

  1. Go to AS management console and select Configureâ†’Service Data Publishing from side panel.
  2. Fill the form.
    1. Tick on Enable Service stats and Enable Activity Service.
    2. Setting up the Stream Definition Configuration
      1. Stream Name : Stream Name can be any string with alpha-numeric characters
      2. Version : Stream Version distinguishes different streams with the same Stream Name. Default version should be 1.0.0 .
      3. Nick Name : This is a user preferred nick name to the Stream Name in alpha-numeric characters
      4. Description : A description describing about the particular stream defined by Stream Name, Stream Version pair. Description should also consists of alpha-numeric characters.
    3. Setting up BAM Credentials.
      1. BAM URL : Enter the IP address of the BAM server. And the port should be the thrift port ( ex: tcp://127.0.0.1:7611 ).
      2. Username: Put the BAM Thrift log-in user name. Default value is admin.
      3. Password: Put the BAM Thrift log-in password. Default value is admin.
    4. Click on Update button.
